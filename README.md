## What This Project Does

SYS-Studio is a FiveM website focused on presenting scripts, offering resource downloads, and supporting resource sales. It helps RP server owners and developers keep content neatly arranged, spotlight available resources, and make it simpler for visitors to browse the scripts they want.

The repository is aimed at roleplay servers and the wider FiveM community. Its job is to display resources in a structured way while still being practical for sharing, browsing, and maintaining script listings as they evolve.

## Script and Content Types Included

- Script presentation pages
- Resource download entries
- Resource sales listings
- Roleplay server resources
- FiveM website sections
- Community-facing resource pages
- Update-ready content blocks
- Reference and documentation areas

## Getting Started

1. Download or clone the repository:
   `git clone https://github.com/matt-taylor2002/sys-studio-script-hub.git

2. Open the project folder:
   `cd sys-studio-fivem-site`

3. Arrange your content within the site structure:
   - place scripts or listings in the main content areas
   - add images or previews where needed
   - keep resource notes and pricing details together

4. Publish the site or preview it locally, then refresh links whenever your resources change.

Example structure for managing content:
- `scripts/` for script listings or resource pages
- `configs/` for site settings and display options
- `examples/` for sample layouts or demo entries
- `docs/` for notes, instructions, and reference material
